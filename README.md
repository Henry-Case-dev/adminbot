# AdminBot — ваш персональный цифровой абьюзер

*Элегантное решение для автоматизации токсичного общения в чатах.*

## 🚀 Возможности

- **Костик**: на каждое сообщение (с настраиваемой вероятностью `KOSTIK_REPLY_PROBABILITY`, по умолчанию 100%) получает случайную фразу из пула из 8 персональных оскорблений. Абсолютно заслуженно.
- **Слава**: многоуровневая система оскорблений. Просто пишет → «пошёл нахуй». Упоминает кучу → «ДАЛБАЕБ». Ляпнул про войну → «трясло ебаное». А ещё каждое 5-е сообщение — гифка. Мы следим за тобой, Слава.
- **Вася**: кто-то пишет «вася» → бот орёт «АДМИН». Пишут «админ» → орёт «ВАСЯ». Шах и мат, лингвисты.
- **Alan_Z**: каждые 10 сообщений бот проявляет крайний интерес к его темам (тренировки, лонгковид, фьючерсы, нейросети, жим дьявола) случайной фразой из пула ~25 вариантов.
- **Слава 2: Возвращение**: заходит в чат → «ДОЛБОЕВ ВЕРНУЛСЯ». Уходит → тишина и облегчение. Дважды в день бот напоминает ему о бренности бытия случайной картинкой из dead_page.

## ⚡ Быстрый старт

```bash
pip install -r requirements.txt
cp .env.example .env  # и впиши туда API_TOKEN
mkdir -p media/dead_page  # накидай туда .jpg и .txt
python bot.py
```

## 🛠 Настройка и кастомизация

### Параметры через `.env`

Все настройки бота управляются через файл `.env`. После изменения перезапусти бота.

| Переменная | По умолчанию | Описание |
|-----------|-------------|----------|
| `API_TOKEN` | *(обязательно)* | Токен Telegram-бота от @BotFather |
| `KOSTIK_USER_ID` | `350803143` | ID Костика |
| `KOSTIK_REPLY_PROBABILITY` | `1.0` | Вероятность ответа Костику (0.0 — никогда, 0.5 — 50%, 1.0 — всегда) |
| `SLAVIK_USER_ID` | `479167456` | ID Славы |
| `ALAN_USER_ID` | `138811255` | ID Alan_Z |
| `ALAN_REPLY_INTERVAL` | `10` | Каждые N сообщений от Alan — ответ |
| `GIF_INTERVAL` | `5` | Каждые N сообщений Славы — гифка |
| `MORNING_HOUR` | `10` | Час утреннего dead-page поста |
| `EVENING_HOUR` | `20` | Час вечернего dead-page поста |

### Как добавить/изменить варианты ответов

Все пулы ответов лежат в соответствующих файлах хендлеров. Каждый пул — это список строк `REPLIES`. Просто добавь новую строку в список.

#### Костик — `handlers/kostik.py`

```python
KOSTIK_REPLIES = [
    "пошёл нахуй кринжатура ебаная",   # ← старый
    "твой новый insult сюда",           # ← добавь так
    "ещё один вариант",                 # ← и так
    # ... сколько угодно
]
```

#### Alan_Z — `handlers/alan.py`

```python
ALAN_REPLIES = [
    "так интересно, продолжай",        # ← старый
    "твоя новая фраза сюда",           # ← добавь так
    # ... любые фразы про тренировки, лонгковид, фьючерсы, нейросети
]
```

#### Слава («КУЧА», «война», catch-all) — `handlers/slavik.py`

Ответы Славы захардкожены в коде хендлеров (`"ДАЛБАЕБ"`, `"трясло ебаное"`, `"пошёл нахуй"`). Чтобы изменить — отредактируй строку в соответствующем хендлере:

```python
# handlers/slavik.py
async def kucha_handler(message: types.Message):
    await message.reply("ТВОЙ НОВЫЙ ОТВЕТ")  # ← меняй здесь
```

#### Вася — `handlers/vasya.py`

Аналогично Славе — ответы захардкожены (`"АДМИН"`, `"ВАСЯ"`). Меняй прямо в файле.

### Как изменить ID пользователей

В `.env`:
```bash
KOSTIK_USER_ID=350803143    # ← поменяй на нужный ID
ALAN_USER_ID=138811255      # ← поменяй на нужный ID
SLAVIK_USER_ID=479167456    # ← поменяй на нужный ID
```

### Как добавить мемные картинки в dead_page

Кидай `.jpg`/`.png` и `.txt` файлы в папку `media/dead_page/`. Бот сам подберёт случайную пару «фото + текст» при отправке dead-page поста.

### После любых изменений

```bash
# Проверь что ничего не сломалось
py -m pytest tests/ -v

# Если все 130 тестов зелёные — перезапускай бота
python bot.py
```

## 🐛 Известные проблемы

- Слава иногда обижается, но всегда возвращается.
- Костик не понимает, за что.
- Alan_Z иногда не отвечает, но это нормально — он ждёт 10-го сообщения.
- При отсутствии интернета уровень токсичности падает до нуля, что недопустимо.
- Бот не умеет оскорблять создателя. Мы работаем над этим.
