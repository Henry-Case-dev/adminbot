# AdminBot — ваш карманный цифровой абьюзер с элементами искусственного интеллекта

*Решение для тех, кто хочет токсичности в чате, но ленится писать сам. Теперь с памятью слона и терпением снайпера.*

**Версия:** v2.8.0 | **Тестов:** 270 | **Эпиков:** 13 (74 задачи)

---

## 🚀 Возможности

### Слава: возвращение блудного долбоёба (F1)
Когда Слава заходит в чат (покинул → вернулся) — бот пишет **«ДОЛБОЕВ ВЕРНУЛСЯ»** и сразу шлёт dead page. Потому что надо же как-то обозначить событие. Каждый раз одно и то же, и каждый раз смешно. Славе — нет.

### Слава: Dead Page (F2 v2)
- Кто-то делает репост из канала **@d_pages** → бот выбирает случайный пост из своего приватного канала и пересылает его в чат. Технология «слышу звон — несу что попало» доведена до совершенства.
- Если приватный канал упал или бот оттуда выгнали → fallback: случайная картинка + текст из папки `media/dead_page/`. Качество контента не гарантируется, но volume присутствует.
- При входе Славы → «ДОЛБОЕВ ВЕРНУЛСЯ» + dead page (если не отключено в настройках, но зачем отключать-то?)

### Слава: чат-реакции (F3/F4)
Трёхуровневая система эскалации абьюза:
- Просто написал → **«пошёл нахуй»** (catch-all, потому что Слава всегда заслуживает)
- Упомянул «куча»/«кучи»/«кучу» → **«ДАЛБАЕБ»** (F4 — прицельный выстрел по грамматике)
- Каждое 5-е сообщение → **GIF** со славным членом (F3 — визуальное подкрепление)

### Слава: военный оповещатель (F5v2 — Epic 10)
Когда Слава пишет сообщение с военными ключевыми словами (дрон, БПЛА, ракета, опасность, тревога, убежище и т.д.) — бот отвечает случайной фразой из пула: «потрясись», «повизжи», «прячься под шконку быстрее», «закрой ушки и считай до десяти», «поплачь». Психологическая поддержка наоборот.

**Новое в v2 (Epic 10):**
- **Caption support**: теперь бот видит угрозу даже в подписях к медиа. Раньше был слеп как крот — теперь зряч как орёл.
- **Channel reposts**: если **любой пользователь** репостит из каналов «ЧП Пермь» или «Радар БПЛА» — бот тоже орёт. Коллективная паника.
- **Расширенный словарь**: 90+ словоформ. Бот знает о войне больше, чем некоторые военкоры.
- **Настраиваемый пул фраз**: `WAR_REPLIES` в `.env`. Хотите «всё пропало» вместо «потрясись»? Да пожалуйста.
- **Детальное логгирование**: каждый чих пишется в Better Stack. Можно потом пересматривать как сериал.

### Alan_Z: автоответчик (F6)
Каждые N сообщений Алана бот проявляет **крайний интерес** к темам: тренировки, лонгковид, фьючерсы, нейросети, жим дьявола. Случайная фраза из пула ~40 вариантов. Маскируется под живого собеседника. Алан до сих пор не понял.

### Alan_Z: приветственное видео (F7)
Когда Alan заходит в чат — бот отправляет случайное видео из `media/leha_greeting/` с подписью `@Alan_Z`. Срабатывает через `ChatMemberUpdated` (основной детект) и `new_chat_members` (fallback для параноиков). Кулдаун 10 секунд — чтобы не заддосить чат радостью.

### Alan_Z: приветствие после сна (F7 v2 — Epic 11) 🆕
**Самая пассивно-агрессивная фича бота.** Если Алан **молчит** в чате дольше `ALAN_SILENCE_GREETING_HOURS` часов, а потом наконец пишет сообщение — бот встречает его приветственным видео. Как будто соскучился. На самом деле — просто фототропизм на движение.

**Как это работает:**
1. Бот запоминает timestamp каждого сообщения Алана в БД (переживает рестарт — память как у слона).
2. Перед каждым новым сообщением вычисляет `elapsed = сейчас − последнее_сообщение`.
3. Если elapsed ≥ N часов → «С ВОЗВРАЩЕНИЕМ!» через случайное видео.
4. Если Алан пишет слишком часто → таймер сбрасывается без приветствия (не заслужил).
5. Общий anti-spam с join-приветствием: если видео только что отправлено при входе — silence не дублирует.

**Зачем это нужно:** потому что наблюдать за реакцией Алана, когда бот «узнаёт» его после долгого отсутствия — бесценно.

| Переменная | По умолчанию | Что делает |
|-----------|-------------|-----------|
| `ALAN_SILENCE_GREETING_HOURS` | `6.0` | Часы молчания, после которых следующее сообщение триггерит видео. `0.5` = 30 мин, `0.0` = фича отключена. Требует перезапуска бота |

### Костик: генератор случайных оскорблений (F8)
На каждое сообщение — оскорбление из пула. Вероятность настраивается: `1.0` = всегда (классика), `0.0` = никогда (режим «Костик в отпуске»). Костик не выбирал эту жизнь, жизнь выбрала его.

### Вася / Админ: лингвистическая симметрия
- Написал «вася» (в любых формах) → **«АДМИН»**
- Написал «админ» → **«ВАСЯ»**

Обратите внимание: это не баг, это зеркало души чата.

### Тестовые команды для админа (Epic 10)
Ручное тестирование без ожидания Славу/Алана:

| Команда | Что делает | Где работает |
|---------|-----------|-------------|
| `/deadpage` | Триггерит dead page | DM — всем, группа — только админу |
| `/alangreet` | Триггерит приветственное видео Алана | DM — всем, группа — только админу |

Сообщение с командой **автоматически удаляется** — админ должен оставаться неуловимым.

## 🖼 Как работают dead page и fallback

### Основной режим (канал жив)
1. Кто-то репостит из **@d_pages** → бот просыпается
2. Идёт в приватный канал, выбирает случайный пост, форвардит в чат
3. Всё одним сообщением, без разрывов. Магия.

### Fallback (канал мёртв / бота выгнали)
1. Бот берёт случайный `.jpg` и случайный `.txt` из `DEAD_PAGE_DIR`
2. Отправляет фото + подпись (до 1024 символов)
3. Если текста больше — второе сообщение с хвостом

Пополнение: кидай `.jpg` и `.txt` в `media/dead_page/`. Profit.

## 🎨 Как изменить текст ответов

Все ответы лежат в файлах хендлеров. Каждый пул — список строк. Добавь строку → расширил пул. Элементарно, Ватсон.

### Костик — `handlers/kostik.py`
```python
KOSTIK_REPLIES = [
    "пошёл нахуй кринжатура ебаная",
    "твой новый insult сюда",        # ← добавь
]
```

### Alan_Z: автоответчик — `handlers/alan.py`
```python
ALAN_REPLIES = [
    "так интересно, продолжай",
    "твоя новая фраза сюда",         # ← добавь
]
```

### Alan_Z: приветственное видео — `media/leha_greeting/`
Кидай `.mp4` (и `.avi`, `.mov`, `.webm`) в папку. Бот сам выберет случайный.

### Слава — `handlers/slavik.py`
Ответы захардкожены прямо в коде:
```python
await message.reply("ДАЛБАЕБ")       # «куча»
await message.reply("трясло ебаное")  # военные слова
await message.reply("пошёл нахуй")    # всё остальное
```

### Вася / Админ — `handlers/vasya.py`
```python
await message.reply("АДМИН")  # «вася»
await message.reply("ВАСЯ")   # «админ»
```

## 📊 Мониторинг (Better Stack)

| Компонент | Что делает | Библиотека |
|-----------|-----------|-----------|
| **Sentry** | Отлов исключений со стектрейсами | `sentry-sdk` |
| **Logtail** | Стриминг логов в реальном времени | `logtail-python` |

### Как смотреть логи
1. [betterstack.com](https://betterstack.com/) → **Logs & Metrics** → **Live tail**
2. Фильтруй по `level:error`, имени модуля, или тексту
3. Все `logger.info/warning/error` из кода прилетают с тегом `source`

### Как смотреть ошибки
1. Better Stack → **Errors** → Issues со стектрейсами
2. Контекст: чат, сообщение, теги. Как чёрный ящик, только полезный.

### Локально (без Better Stack)
Нет `SENTRY_DSN` и `LOGTAIL_SOURCE_TOKEN` в `.env` → логи только в консоль. Всё работает.

## 🐛 Известные нюансы

- `DEAD_PAGE_COOLDOWN_SECONDS=0` → спам на каждый репост. Весело, но чревато баном.
- Слава иногда обижается, но всегда возвращается. Это не баг, это фича личности.
- Костик не понимает, за что. И не поймёт.
- Без интернета уровень токсичности падает до нуля. Техническое ограничение вселенной.
- Бот должен быть **админом** приватного канала из `DEAD_PAGE_RELAY_CHANNEL_ID`.
- Приветственное видео (F7) срабатывает при **переходе статуса**. Если Алан уже в чате при старте бота — используй `/alangreet`.
- **F7v2 (silence greeting):** таймер хранится в БД и **переживает рестарт бота**. Если бот перезапустить, Алан не «обнулится» — elapsed считается от реального последнего сообщения.
- При поиске постов в relay-канале используется прогрессивный поиск. Если канал разросся до 2000+ сообщений — бот справится, но не мгновенно.
